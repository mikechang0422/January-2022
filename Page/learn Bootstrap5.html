格線的另一個重點:斷點判斷
bootstrap都是用>=下去判斷的
不寫 0-576px (手機)
sm >=576px (手機-橫)
md >=768px (Ipad-直)
lg >=992px (Ipan-橫)
xl >=1200px (13吋筆電)
xxl >=1400px (15.6吋筆電)
定寬容器
container，除了 < 576的情況都是100%
其餘的會根據你設定成-sm & -md等依據上面的尺寸保留大約3%的左右留白
當設成col時會平均分配排版，如果不會換行
當設成col-12 col-ms-6 col-md-4 col-lg-3 col-xl-2時 col-xxl-1
<=576為1欄
>=576為2欄
>=768為3欄
>=992為4欄
>=1200為6欄
>=1400為12欄

可以在row設定主軸對齊方向

1~12

d = display

block = d-block

inline = d-inline

table = d-table

flex = d-flex

hidden = d-none

size = xs md lg xl xxl

for example

d-none d-md-block
正常隱藏但在md以上已block顯示

d-block d-md-none
正常顯示但大於md消失

d-block d-md-inline
正常block但大於md則改inline

麻煩一點的
d-block d-md-none d-lg-block
正常block / md~lg之間消失 / 大於lg則block

offset就是設定margin-left
一樣有1-12
剩下的就是依照你的等分進行margin
範例為 col-3 與 offset-3
第三個正常是在第九7-8-9等分
但因為多了offset-3
則會變成10-11-12的位置

如果你要做文字壓在圖片上屬於交錯式的設計，也可以利用offset來製作
像是第一個設成col-8，第二個col-8 offset-4 則會一左一右，然後再做個margin-top:-100就可以重疊了
